添加栏目
参数 类型 说明
action string menu_add
menuname string 栏目名称
upid int 上级栏目 一级栏目填0
yc int 0=显示 1=隐藏
languageID int 语言ID 默认为0 0就是Chinese 支持哪些语言可通过api_web.php?action=getLanguage 获得 当前页的语言ID可通过全局变量 window.LANGUAGEID = '0'; 获得
调用方式:HTTP POST
URL:http://www.rich2020.com/block/api_edit.php
DEMO:http://www.rich2020.com/block/api_edit.php?action=menu_add&menuname=首页&upid=0&yc=0&languageID=0
返回结果:JSON字符串
注释:成功会返回所添加栏目的ID
----------返回的结果有如下几种：----------
{"status":0,"message":"栏目类别丢失"}
{"status":0,"message":"此栏目名称已存在"}
{"status":0,"message":"upid参数有误"}
{"status":0,"message":"language参数不合法"}
{"status":0,"message":"保存失败"}
{"status":1,"message":"ok","id":123}
